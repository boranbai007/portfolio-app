# Salamat Portfolio

A small portfolio project with two core files:

- `index.html` — the main static website
- `portfolio_marimo.py` — the Marimo version of the UI (no HTML strings in Python, only `mo.*` components)

## What `portfolio_marimo.py` does

1. Renders the portfolio in Marimo using components (`mo.md`, `mo.image`, `mo.hstack`, `mo.vstack`, `.style(...)`).
2. Creates `output.html` as an exact byte-for-byte copy of `index.html`.

## Run

From the project root:

```bash
python portfolio_marimo.py
```

After running, this file is created/updated:

- `output.html`

## Dependencies

You need Python 3.13+ and `marimo`:

```bash
pip install marimo
```

## Structure

- `index.html` — reference HTML version of the page
- `output.html` — copy of `index.html`, generated by the script
- `portfolio_marimo.py` — Marimo UI + copy-to-`output.html` logic
- `assets/` — portfolio images

## How to make changes

1. If you want to change the final website, edit `index.html`.
2. If you want to change the Marimo presentation, edit `portfolio_marimo.py`.
3. After edits, run:

```bash
python portfolio_marimo.py
```

to refresh `output.html`.
